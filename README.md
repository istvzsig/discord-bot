# Discord Bot Framework

A scalable Discord.js bot framework.

## Features

- Slash Commands
- Event Handler
- Command Handler
- Modular Structure

## Setup

```bash
npm install
```

**Create .env:**

```bash
TOKEN=
CLIENT_ID=
GUILD_ID=
```

**Deploy commands:**

```bash
node src/deploy/deployCommands.js
```

**Run Bot:**

```bash
npm run dev
```
